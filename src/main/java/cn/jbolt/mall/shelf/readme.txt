======电商模块======
==货架管理模块==
主要就是用户可以创建多个货架，比如创建一个新年专用的货架，
创建一个端午节专用的货架，一个货架相当于小程序的一个首页样式
不同的货架每次只能有一个是开启有效的，不同时间段使用不同的货架展示营销

每个货架都是由若干个货架元素组成
==货架元素==
分为轮播图 商品楼层 商品分组 LOGO 活动图片区等
每种元素单独表结构

最终一个货架就是一个首页，整体缓存JSON
